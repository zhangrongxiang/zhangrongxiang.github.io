逆序输出数字

发表于2023-09-06|更新于2024-05-17|C语言
|阅读量:

输入一个数字后逆序输出之。
用数列吗？也可以。

更为巧妙的是，利用了取余运算。
因为整数取余给整型，仍会强制转换为整型，因此不断除以10并取余。

#include<stdio.h>

int main(){
    int inp = 0;
    scanf("%d",&inp);

    while(inp%10!=0){
            printf("%d",inp%10);
            inp = inp/10;
    }
    return 0;
}
